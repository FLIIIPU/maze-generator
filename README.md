<-- Maze generator-->

<-- features -->
-dual grid support: generates mazes on square or hexagonal grids
-procedural generation: uses dfs to generate solvable labyrinths
-braid maze: algorithm that remove dead-ends based on user input
-exploration: allows user to move through tiles by clicking
-visualization: uses sfml to show everything in real time

<--tech-->
-language: c++ 20
-graphics: SFML
-IDE: Visual Studio 2022

<--instalation-->
1. ensure you have the SFML library installed and configured in your environment
2. clone this repository:' git clone https://github.com/FLIIIPU/maze-generator'
3. build and run the project

you can also find a pre-compiled version in the [releases](https://github.com/FLIIIPU/maze-generator/releases)
<--additional info-->
this is a high school project, currently in development
additional features will be added in future
whole implementation of square and hex will be updated into better ones
